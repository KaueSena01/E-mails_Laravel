## What is mailtrap?

* Mailtrap is a free tool to test sending emails. This tool is extremely useful in a local development environment, because the email sending is centralized and through the tool it is much easier to analyze the email content.

## Contact page

* Send a message through the contact page and the message will be registered and sent to the email used in the Mailtrap platform service.
